# Todo List App

A modern and responsive Todo List web application built with HTML, CSS, and JavaScript.

This project helps users manage daily tasks with features like task creation, editing, completion tracking, search, calendar filtering, multilingual support, notifications, and animated UI effects.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks with animation
- Mark tasks as completed
- Progress bar for completed tasks
- Task statistics
- Search tasks instantly
- Filter tasks by date
- Priority levels:
  - Low
  - Medium
  - High
- Categories:
  - Work
  - Study
  - Gym
  - Personal
- Local storage support
- Browser notifications
- Confetti celebration effect
- English / Georgian language switch
- Fully responsive design
- Modern glassmorphism UI

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- Notification API

## Main Functionalities

### Task Management

Users can:

- Add tasks
- Edit tasks
- Delete tasks
- Complete tasks
- Organize by priority and category

---

### Language Support

The app supports:

- English 🇺🇸
- Georgian 🇬🇪

Users can switch languages instantly using the language toggle.

---

### Data Persistence

Tasks and username are stored using `localStorage`, so data remains after refreshing the page.

---

### Notifications

When a task is completed:

- Browser notification appears
- Confetti animation plays
